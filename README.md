# Number Guessing Game

A simple Python project where the computer selects a random number and the user has to guess it.

## How it works:
- Computer generates a random number
- User enters a guess
- Program gives hints: "Too high" or "Too low"
- Game continues until the correct number is guessed

## Technologies Used:
- Python

## Features:
- Simple command-line game
- Interactive user input
- Beginner-friendly logic practice

## Purpose:
This project was created to practice basic Python concepts like:
- Loops
- Conditional statements
- User input handling
